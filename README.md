# Task 5 – Capstone Integration & Portfolio Finalization

## Objective
The objective of this project is to integrate data analysis, visualization, and reporting into a professional sales dashboard using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI
- Microsoft Excel

## Dataset
ApexPlanet_DataAnalytics_Dataset.xlsx

## Project Description
- Loaded and cleaned the sales dataset.
- Performed data analysis using Python.
- Calculated Total Sales, Total Quantity, Total Customers, and Total Orders.
- Created an interactive Power BI dashboard.
- Added cards, charts, and slicers for better visualization.

## Dashboard Visuals
- Total Sales Card
- Total Quantity Card
- Clustered Bar Chart
- Line Chart
- Pie Chart
- Donut Chart
- Treemap
- Customer ID Slicer
- Order Date Slicer

## Outcome
The dashboard provides a clear overview of sales performance, customer information, product categories, and sales trends. It helps users analyze business performance through interactive visualizations.
